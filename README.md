# ☒CHIP IX03
<img src="extras/IX03 V0.5.0.JPG" width="35%" height="auto" align="right">
UART to ☒BUS breakout based on the SC16IS750IBS.

This module can be purchased on [Xinabox CC](https://xinabox.cc/products/IX03/).

## About the IX03
This ☒CHIP is based on the SC16IS750IBS chipset, which is SC16IS750 is a slave I2C-bus/SPI interface to a single-channel high performance UART.

More information on the SC16IS750IBS can be found in the [datasheet](https://www.nxp.com/docs/en/data-sheet/SC16IS740_750_760.pdf?).
